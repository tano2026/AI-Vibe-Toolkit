# Script Video 62 — YouTube MCP (mcp-youtube)

## Thông tin
- Tool/Repo/Skill liên quan: /mcps/mcp-youtube.md
- Platform: TikTok / YouTube Shorts
- Thời lượng dự kiến: ~45 giây

## Hook (3 giây đầu)
"Mày đang ngồi xem hết video 20 phút chỉ để lấy 1 ý chính? Bỏ luôn đi."

## Script voiceover (ElevenLabs-ready)
[Đoạn 1 — vấn đề] Vibe coder nào cũng từng vậy: thấy video review tool mới, bấm vào xem, 20 phút sau vẫn chưa tới đoạn quan trọng.

[Đoạn 2 — giải pháp] Có 1 MCP free tên mcp-youtube, gắn vào Claude là xong. Claude tự đọc transcript video qua yt-dlp, không cần YouTube API key luôn.

[Đoạn 3 — demo] Cách dùng: cài yt-dlp, dán đúng 5 dòng JSON vào file config Claude, restart app. Xong, dán link video vào chat, gõ "summarize this", Claude trả tóm tắt trong vài giây.

[Đoạn 4 — kết + CTA] Lưu ý video càng dài transcript càng dễ tràn, và video không có caption thì bó tay. Nhưng để research nhanh thì cực đáng. Link full setup ở bio.

## Ghi chú quay (OBS)
- Cảnh 1: quay màn hình đang xem video YouTube dài, tua nhanh timeline để thấy "lâu"
- Cảnh 2: quay paste đoạn JSON config vào claude_desktop_config.json
- Cảnh 3: quay restart Claude Desktop, zoom vào icon hammer hiện tool mới
- Cảnh 4: quay dán link video vào Claude, Claude trả tóm tắt — zoom vào câu trả lời

## Caption/Sub note (CapCut)
Highlight chữ "không cần API key", "yt-dlp", "vài giây". Cắt nhịp nhanh ở đoạn chuyển từ "20 phút" sang "vài giây" để tạo tương phản rõ.

## Thumbnail idea (Canva)
Chia đôi màn hình: bên trái icon đồng hồ to + chữ "20 PHÚT" nền đỏ, bên phải icon Claude + chữ "5 GIÂY" nền vàng. Font to, tương phản mạnh.
