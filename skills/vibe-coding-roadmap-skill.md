## Roadmap Tóm Tắt (4 Giai Đoạn)

### Giai Đoạn 1 — Nền Tảng (Tuần 1-2)
```
□ Hiểu AI là gì, LLM là gì
□ Cài Claude Desktop + Claude Code
□ Học cách viết prompt hiệu quả
□ Làm quen với terminal cơ bản
□ Git cơ bản (clone, commit, push)
```

### Giai Đoạn 2 — Công Cụ (Tuần 3-4)
```
□ Claude Code — AI pair programmer
□ Cursor hoặc Windsurf — AI code editor
□ MCP cơ bản: Context7, Brave Search, Firecrawl
□ GitHub — lưu và deploy code
□ Vercel/Netlify — deploy web app
```

### Giai Đoạn 3 — Build (Tháng 2-3)
```
□ Build web app đầu tiên (HTML/CSS/JS + AI)
□ Build Python script (automation)
□ Build API đơn giản (FastAPI + Claude)
□ Deploy lên cloud
□ Connect với database (SQLite → PostgreSQL)
```

### Giai Đoạn 4 — Ship & Scale (Tháng 3+)
```
□ Add authentication
□ Add payment (Stripe)
□ Monitor với logging
□ CI/CD pipeline
□ Build AI agent đầu tiên
```

## Stack Gợi Ý Theo Goal
