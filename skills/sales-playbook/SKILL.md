# Sales Playbook — Battle Cards & Scripts

Từ ai-marketing-skills repo.

## Tools
```bash
python "D:/AI Store/digital-marketing-repos/ai-marketing-skills/sales-playbook/generate_playbook.py" --product "Your product name" --industry saas --competitors "Competitor A, Competitor B"
```

## Output
- **Battle cards**: per-competitor positioning + counter-messaging
- **Objection handling**: top 10 objections + scripted responses
- **Discovery questions**: role-specific question banks
- **Call scripts**: opening → discovery → demo → close
- **Competitive positioning**: feature comparison + differentiation
