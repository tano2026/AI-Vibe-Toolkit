## Đây Là Gì

X/Twitter research agent cho **Claude Code** và **OpenClaw**. Wrap X API thành CLI nhanh — AI agent search tweets, pull threads, monitor accounts, get sourced research mà không cần viết curl commands.

---

## Features

- **Search** với engagement sorting, time filtering, noise removal
- **Quick mode** — cheap, targeted lookups
- **Watchlists** — monitor accounts liên tục
- **Cache** — tránh repeat API charges
- **Cost transparency** — mỗi search show cost
