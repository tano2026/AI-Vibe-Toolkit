## 31 Skills — Theo Pipeline

### 🔍 SEARCH & DISCOVER
| Skill | Làm gì |
|-------|--------|
| `literature-search` | Tìm papers theo topic — Semantic Scholar, arXiv |
| `literature-review` | Review + tổng hợp literature |
| `github-research` | Research GitHub repos cho research topics |
| `novelty-assessment` | Đánh giá novelty của idea so với SOTA |

### 💡 IDEATION & PLANNING
| Skill | Làm gì |
|-------|--------|
| `idea-generation` | Generate research ideas từ gap analysis |
| `research-planning` | Lên kế hoạch research từ đầu đến cuối |
| `experiment-design` | Design experiments + ablation studies |
| `atomic-decomposition` | Break research task thành atomic steps |

### 🔬 EXECUTION
| Skill | Làm gì |
|-------|--------|
| `experiment-code` | Viết code cho experiments |
| `paper-to-code` | Implement paper từ mô tả |
| `math-reasoning` | Derive math formulas + proofs |
| `symbolic-equation` | LaTeX equations từ mô tả |
| `data-analysis` | Phân tích experimental results |
| `algorithm-design` | Design thuật toán mới |
| `code-debugging` | Debug experiment code |

### 📝 WRITING
| Skill | Làm gì |
|-------|--------|
| `paper-writing-section` | Viết từng section (intro, method, exp, conclusion) |
| `related-work-writing` | Viết related work với proper citations |
| `rebuttal-writing` | Viết rebuttal cho reviewer comments |
| `paper-revision` | Revise paper theo feedback |
| `self-review` | AI review paper trước khi submit |
| `citation-management` | Manage + format citations |
| `backward-traceability` | Track claims back to sources |

### 📊 VISUALIZATION
| Skill | Làm gì |
|-------|--------|
| `figure-generation` | Generate figures + plots |
| `table-generation` | Generate comparison tables |
| `excalidraw-skill` | Diagrams với Excalidraw |

### 📚 OUTPUT
| Skill | Làm gì |
|-------|--------|
| `paper-assembly` | Assemble full paper từ sections |
| `paper-compilation` | Compile LaTeX paper |
| `latex-formatting` | Format theo venue style (NeurIPS, ICML...) |
| `survey-generation` | Generate survey/review paper |
| `slide-generation` | Generate presentation slides từ paper |

## Token-Efficient Research Workflow

```
Idea → literature-search (targeted) → novelty-assessment
     → research-planning → experiment-design
     → experiment-code → data-analysis
     → paper-writing-section (per section) → self-review
     → paper-assembly → latex-formatting → slide-generation
```

Mỗi skill chỉ làm 1 việc → token focused, không waste.
