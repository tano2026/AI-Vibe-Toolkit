## 🎯 Roadmap này dạy gì

Không phải học code theo nghĩa truyền thống — mà học **cách dùng AI để build product:**

**Phase 1 — Foundation**
- Hiểu AI coding tools hoạt động thế nào
- Prompt engineering cơ bản
- Setup môi trường (Cursor, Claude Code, hoặc bolt.new)

**Phase 2 — Build First Project**
- Clone template có sẵn thay vì từ đầu
- Vòng lặp: describe → AI code → review → iterate
- Deploy lên Vercel/Netlify

**Phase 3 — Ship Product**
- Stack chuẩn 2026: Next.js + Supabase + Tailwind + Vercel
- Authentication, payments, database
- From idea → live product

**Phase 4 — Level Up**
- Đọc code AI viết để học
- Debug khi AI sai
- MCP servers và AI agents
