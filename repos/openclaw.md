## 🎯 OpenClaw là gì

Hình dung thế này: **Claude Code nhưng chạy 24/7, tự động, kết nối mọi app mày dùng.**

- Mày nhắn Telegram → agent xử lý → tự làm việc → báo kết quả
- Kết nối 50+ apps: WhatsApp, Telegram, Slack, Discord, iMessage, Gmail...

- Có 13,700+ skills từ ClawHub marketplace
- Data không rời máy mày — privacy hoàn toàn

## ⚡ Cài trong 3 bước

```bash
# Bước 1: Cài Node.js 24 từ nodejs.org

# Bước 2: Cài OpenClaw
npm install -g openclaw

# Bước 3: Setup tự động
openclaw onboard
# Làm theo hướng dẫn → chọn LLM → kết nối Telegram/WhatsApp → Done
```
