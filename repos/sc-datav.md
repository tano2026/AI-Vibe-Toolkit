## 🎯 Đây là gì

Kiểu dashboard mà mày hay thấy trong phim:
- Màn hình tối, neon xanh
- Bản đồ 3D xoay được
- Số liệu nhảy real-time
- Biểu đồ bay qua bay lại

**sc-datav build cái đó bằng Three.js + Vue3** — mà mày không cần biết 3D graphics. Clone về, thay data của mày vào, có ngay dashboard xịn.

## 💡 Dùng để làm gì thực tế

- Portfolio cá nhân siêu ấn tượng
- Dashboard cho sếp xem số liệu kinh doanh — wow factor cực cao
- Demo product cho khách hàng
- Visualization cho data project

## ⚡ Bắt đầu nhanh

```bash
# Clone về
git clone https://github.com/knight-L/sc-datav

# Cài dependencies
cd sc-datav
npm install

# Chạy thử
npm run dev

# Mở browser → localhost:3000 → thấy ngay dashboard 3D
```

**Thay data của mày vào:**
Tìm file data trong `/src/data/` → thay số liệu → dashboard tự update.
