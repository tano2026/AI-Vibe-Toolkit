# 🧰 AI Vibe Toolkit

> Kho tổng hợp MCP, Skills, Repos và Tools hay nhất cho **vibe coders** — được test thực tế, viết lại bằng tiếng người, dùng được ngay.

Tao là người mới bắt đầu với AI tools. Kho này là những gì tao học được trong quá trình khám phá — không pretend là expert, chỉ chia sẻ những gì thực sự hữu ích.

---

## 📁 Cấu trúc kho

| Folder | Nội dung |
|--------|----------|
| [`/mcps`](./mcps/) | MCP Servers hay — cài được, dùng được ngay |
| [`/skills`](./skills/) | Skills cho Claude và AI agents |
| [`/repos`](./repos/) | GitHub repos đáng học và tham khảo |
| [`/stacks`](./stacks/) | Combo tools cho từng mục đích cụ thể |
| [`/content`](./content/) | Script video, bài đăng, nội dung sẵn dùng |

## 🚀 Dành cho ai?

- Vibe coders mới bắt đầu
- Dev không chuyên muốn dùng AI tools hiệu quả
- Người không có background kỹ thuật nhưng muốn build stuff với AI

---

## 📌 Cách dùng kho này

1. Vào folder phù hợp với nhu cầu
2. Đọc file `.md` — có hướng dẫn setup từng bước
3. Copy-paste và chạy
4. Nếu gặp lỗi → tạo Issue hoặc liên hệ
